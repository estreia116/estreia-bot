### 📱 METODO 1: TERMUX (Android)
*Apri Termux e incolla questi comandi uno alla volta:*

1.  **Preparazione:** 
    ```bash
    pkg update && pkg upgrade -y
    pkg install git nodejs -y
    ```
2.  **Clone del Bot:**
    ```bash
    git clone [https://github.com/aahmed2542011-png/Stellina-bot-.git](https://github.com/aahmed2542011-png/Stellina-bot-.git)
    cd Stellina-bot-.git
    ```
3.  **Avvio:**
    ```bash
    npm install
    npm start
    ```

---

### 💻 METODO 2: VPS (Linux/Ubuntu)
*Per chi vuole il bot attivo 24 ore su 24.*

1.  **Installazione Core:** Assicurati di avere Node.js e Git sulla VPS.
2.  **Persistenza con PM2:**
    ```bash
    npm install -g pm2
    pm2 start index.js --name "endy-bot"
    pm2 save
    pm2 startup
    ```
3.  **Monitoraggio:** Usa `pm2 logs Estreia-bot` per visualizzare il QR Code e i log.
